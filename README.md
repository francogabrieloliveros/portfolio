## How to create a GitHub page

1. Create a GitHub repository containing index.html on the root folder.
2. In the repository settings, find the "Pages" section on the left nav.
3. Set the source to "Deploy from a branch".
4. Select the branch you want to deploy from.
5. Hit "Save" and wait a few minutes until the site url is ready.
